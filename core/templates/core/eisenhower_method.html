Как использовать матрицу на практике

1

Составьте список всех задач

Запишите всё, что вам нужно сделать - рабочие задачи, личные дела, долгосрочные цели.

2

Классифицируйте по важности и срочности

Для каждой задачи задайте два вопроса: "Это важно для моих целей?" и "Это нужно сделать срочно?"

3

Распределите по квадрантам

Поместите каждую задачу в соответствующий квадрант матрицы.

4

Действуйте согласно стратегии

Выполняйте задачи в порядке приоритета квадрантов: 1 → 2 → 3 → 4.
